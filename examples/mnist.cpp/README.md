# MNIST Example for GGML

This is a test sample of ggml.cpp on running a simple GGML mnist inferencing.

The model is given. 

## Running the example


```bash
# Build
git clone git@github.com:umarrudy/ggml.cpp.git
cd ggml.cpp/example
make

# Run the MNIST model
./main models/mnist/ggml-model-f32.bin models/mnist/t10k-images.idx3-ubyte
```
